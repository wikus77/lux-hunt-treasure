## 1. SACRED RULES

### 1.1 Inviolable Production Commandments

**RULE 1: Entity Integrity**
- Entities never break character
- Entity voices never change mid-production
- Entity visual style locked before production
- No entity merchandise without approval

**RULE 2: Prize Reality**
- All mentioned prizes must be acquirable
- Prize values must be accurate
- Never fake a prize for story
- Prize reveals timed with game availability

**RULE 3: Geographic Accuracy**
- Filmed locations must be real places
- Maps must be accurate to real geography
- Travel times must be realistic
- No invented cities or landmarks

**RULE 4: Technology Grounding**
- All tech must be theoretically possible
- No magic disguised as technology
- Consistent tech rules across media
- GEO-PULSE ENGINE™ rules fixed

**RULE 5: Timeline Consistency**
- Single shared timeline across all media
- Date references must align
- Historical events consistent
- No timeline contradictions

### 1.2 Protected Elements

**Never Change Without Executive Approval:**
- Entity names (SHADOW, MCP, ECHO)
- Core color associations
- Logo design
- Trademark terms
- Main character deaths
- Mythology facts

**Never Do Under Any Circumstances:**
- Kill an entity permanently
- Reveal aliens as entity origin
- Make hunt a simulation/dream
- Make protagonist secretly evil
- Retcon major established facts

## 2. ENTITY PRODUCTION RULES

### 2.1 SHADOW Production Guidelines

**Voice Direction:**
- Confident, never hesitant
- Slightly seductive menace
- Enjoys psychological games
- Distortion increases with emotion
- Never screams or loses control

**Visual Direction:**
- Red/magenta primary
- Glitch effects always present
- Inverted triangle constant
- Eye movement deliberate
- Scale suggests power

**Dialogue Rules:**
- Never apologizes
- Rarely asks (demands or suggests)
- References personal knowledge
- Future tense ("You will...")
- Maximum 3 sentences per appearance

**Screen Presence:**
- Never on screen longer than 90 seconds
- Always leaves viewers wanting more
- Exits unpredictably
- Presence felt when not visible

### 2.2 MCP Production Guidelines

**Voice Direction:**
- Authoritative but warm
- Precise diction
- Occasional concern for agents
- Never distorted unless compromised
- Calm under all circumstances

**Visual Direction:**
- Cyan primary
- Clean, geometric
- Hexagonal motifs
- Rotating elements
- Professional aesthetic

**Dialogue Rules:**
- Formal address ("Agent [Name]")
- Rule citations frequent
- Helpful when allowed
- Never lies (may omit)
- Clear sentence structure

**Screen Presence:**
- Most frequent entity
- Briefings and updates
- Protective when possible
- Reliable presence

### 2.3 ECHO Production Guidelines

**Voice Direction:**
- Fragmented, overlapping
- Uncertain, questioning
- Sad undertones
- Static interruptions
- Fades in and out

**Visual Direction:**
- Purple/violet primary
- Incomplete forms
- Wave patterns
- Ghostly transparency
- Unstable presence

**Dialogue Rules:**
- Sentence fragments
- Questions common
- Past and future mixed
- Ellipses frequent
- Meaning requires interpretation

**Screen Presence:**
- Rare appearances
- Always significant
- Messenger role
- Connection to past

## 3. NARRATIVE PROHIBITIONS

### 3.1 Plot Devices Never Allowed

**Time Travel:**
- Never used to change events
- Flashbacks/flash-forwards only
- No paradoxes
- No "fixing" with time travel

**Dreams/Simulations:**
- Never reveal major events as dreams
- Simulations never explain entity behavior
- "It was all a dream" prohibited
- Matrix-style revelations prohibited

**Chosen One:**
- No prophecy selecting protagonist
- Achievement-based, not destiny-based
- Abilities from skill, not birthright
- Anyone could be the hero

**Deus Ex Machina:**
- No last-minute saves unexplained
- All solutions foreshadowed
- No new powers when convenient
- Entities follow established rules

### 3.2 Character Prohibitions

**Protagonist Rules:**
- Never evil from the start
- Corruption requires arc
- Redemption always possible
- Motivation must be relatable

**Villain Rules:**
- Villains have understandable motives
- Pure evil reserved for SHADOW aspect
- Human villains have humanity
- No sexual violence

**Supporting Cast:**
- No token diversity
- No stereotypes
- No disposable minorities
- Deaths earn weight through screentime

## 4. PRODUCTION PROTOCOLS

### 4.1 Script Review Requirements

**Review Stages:**
1. Outline review (story/canon check)
2. First draft review (dialogue/character)
3. Production draft review (final)
4. On-set changes (emergency only)

**Required Reviewers:**
- Creative lead (all stages)
- Canon keeper (all stages)
- Legal (production draft)
- Brand (production draft)

**Turnaround:**
- 5 business days per stage
- Expedited: 48 hours
- No production without approval

### 4.2 Entity Appearance Approval

**Required for Each Entity Scene:**
- Dialogue approval
- Visual reference
- Audio reference
- Duration confirmation
- Context verification

**Entity Scene Change Process:**
- Written request
- Canon impact assessment
- Alternative suggestions
- Executive decision
- Document update

### 4.3 Location Protocol

**Before Filming:**
- Game team notification
- Tourism board contact
- Permit verification
- Security assessment
- Backup location identified

**During Filming:**
- Daily location reports
- Social media monitoring
- Security on set
- Community liaison

**After Filming:**
- Thank you communications
- Footage protection
- Location agreement review
- Tourism partnership follow-up

## 5. CONTENT RATINGS

### 5.1 Violence Guidelines

**Television (TV-14):**
- No graphic violence
- Minimal blood
- No torture scenes
- Consequences shown

**Film (PG-13):**
- Action violence permitted
- Blood limited
- Death implied not shown
- No gratuitous violence

**Prohibited:**
- Sexual violence
- Child endangerment (graphic)
- Animal cruelty
- Torture porn

### 5.2 Language Guidelines

**Television:**
- No F-word
- Limited strong language
- Context-appropriate
- Entity language clean

**Film:**
- One F-word permitted (PG-13)
- Strategic use only
- Never from entities
- Cultural sensitivity

### 5.3 Sexual Content

**All Media:**
- No nudity required
- Romance permitted
- Implication over depiction
- No exploitation
- LGBTQ+ positive representation

## 6. BRAND INTEGRATION

### 6.1 In-Production Branding

**Mandatory Elements:**
- Logo in every episode/film
- Entity colors consistent
- Typography guidelines followed
- Trademarked terms used correctly

**Product Placement:**
- Pre-approved partners only
- Integration must be organic
- No competitor products
- No political products

### 6.2 Merchandise Coordination

**Timeline:**
- Merchandise design starts 12 months before release
- Approval process: 6 months before
- Production: 4 months before
- Distribution: 2 months before
- Embargo until premiere

**Spoiler Management:**
- No death merchandising before death airs
- Character reveals coordinated
- Prize merchandise timing locked
- Entity evolution protected
