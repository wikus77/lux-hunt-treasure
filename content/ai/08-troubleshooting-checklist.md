# Troubleshooting Checklist

## General Issues

### App Won't Load
- [ ] Check internet connection (WiFi/mobile data)
- [ ] Try refreshing the page (Ctrl+R or Cmd+R)
- [ ] Clear browser cache and cookies
- [ ] Try incognito/private browsing mode
- [ ] Test in different browser
- [ ] Check if browser is supported (Chrome, Safari, Firefox, Edge)
- [ ] Verify JavaScript is enabled
- [ ] Disable browser extensions temporarily

### Slow Performance
- [ ] Close unnecessary browser tabs
- [ ] Clear browser cache
- [ ] Check available device storage
- [ ] Restart browser
- [ ] Restart device
- [ ] Check for browser updates
- [ ] Test on WiFi instead of mobile data
- [ ] Check for M1SSION service status updates

## Account & Authentication

### Can't Log In
- [ ] Verify email/password are correct
- [ ] Check for typos (case-sensitive)
- [ ] Try "Forgot Password" flow
- [ ] Check email spam folder for reset link
- [ ] Ensure email is verified (check inbox)
- [ ] Try different browser
- [ ] Clear cookies and try again

### Email Not Received
- [ ] Check spam/junk folder
- [ ] Verify email address is correct
- [ ] Add support@m1ssion.eu to contacts
- [ ] Wait 10-15 minutes (delivery delay)
- [ ] Try resending
- [ ] Check email provider isn't blocking
- [ ] Use alternative email address

### Account Locked
- [ ] Wait 30 minutes and try again
- [ ] Contact support with account details
- [ ] Verify you're not using VPN (may trigger security)
- [ ] Check for suspicious activity notification

## Push Notifications

### Not Receiving Notifications
- [ ] Check notification permission (should be "Allowed")
- [ ] Verify push enabled in M1SSION settings
- [ ] Check device notification settings
- [ ] Ensure Do Not Disturb is off
- [ ] Check battery saver isn't blocking notifications
- [ ] Verify outside quiet hours (21:00-08:00)
- [ ] Check you haven't hit daily limit (5 per day)
- [ ] Try resubscribing to notifications

### Notification Permission Denied
- [ ] Reset site permissions in browser
- [ ] Visit Settings → Notifications → Enable
- [ ] Clear browser data and try again
- [ ] On iOS: Settings → Safari → Advanced → Website Data → Remove
- [ ] On Android: Chrome Settings → Site Settings → Notifications

## Location Services

### Location Not Working
- [ ] Check browser location permission
- [ ] Check device location services enabled
- [ ] For iOS: Settings → Privacy → Location Services → Safari
- [ ] For Android: Settings → Location → On
- [ ] Ensure not using VPN (affects accuracy)
- [ ] Try outdoors for better GPS signal
- [ ] Restart location services
- [ ] Grant "Precise Location" permission (iOS 14+)

### Inaccurate Location
- [ ] Move to open area (away from tall buildings)
- [ ] Wait for GPS to stabilize (can take 30 seconds)
- [ ] Check location accuracy setting is "High"
- [ ] Disable battery saver (can reduce accuracy)
- [ ] Ensure WiFi is on (assists GPS)
- [ ] Restart device

## Gameplay Issues

### Clues Not Appearing
- [ ] Check subscription tier allows access
- [ ] Verify mission is active
- [ ] Check mission start date/time
- [ ] Refresh the page
- [ ] Log out and log back in
- [ ] Contact support if issue persists

### BUZZ Not Working
- [ ] Enable location services
- [ ] Check BUZZ count in profile
- [ ] Verify not in cooldown period
- [ ] Ensure within reasonable distance of target
- [ ] Try moving closer to suspected area
- [ ] Check for mission-specific BUZZ limits

### Final Shot Attempts Blocked
- [ ] Check daily attempt limit (2 per day per mission)
- [ ] Wait until next day (resets at midnight)
- [ ] Verify subscription tier allows attempts
- [ ] Contact support if incorrectly limited

## Payment Issues

### Payment Failed
- [ ] Verify card details are correct
- [ ] Check card has sufficient funds
- [ ] Try different payment method
- [ ] Disable browser autofill
- [ ] Check card isn't expired
- [ ] Contact bank (may be blocking transaction)
- [ ] Try PayPal or alternative method
- [ ] Contact support with error code

### Subscription Not Activating
- [ ] Wait 5-10 minutes for processing
- [ ] Check email for confirmation
- [ ] Verify payment went through (check bank)
- [ ] Log out and back in
- [ ] Check subscription status in Account page
- [ ] Contact support with transaction ID

## Progressive Web App (PWA)

### Can't Install PWA
- [ ] Verify browser supports PWA (Chrome, Edge, Safari 16.4+)
- [ ] Look for install prompt in browser
- [ ] Try Settings menu → "Add to Home Screen"
- [ ] Ensure HTTPS connection (required for PWA)
- [ ] Clear browser cache and retry
- [ ] Check device storage has space

### PWA Not Updating
- [ ] Close and reopen app completely
- [ ] Clear app cache (Settings → Storage)
- [ ] Uninstall and reinstall PWA
- [ ] Check for system updates
- [ ] Try uninstalling and using browser version

## Data & Privacy

### Want to Delete Account
- [ ] Go to Settings → Account → Delete Account
- [ ] Or contact support with deletion request
- [ ] Allow 30 days for complete removal
- [ ] Download data first if needed (can't recover later)

### Export My Data
- [ ] Settings → Privacy → Export Data
- [ ] Wait for email with download link
- [ ] Link expires after 7 days
- [ ] Data in JSON format

## Still Having Issues?

- Email: support@m1ssion.eu
- Include: Error message, device info, steps to reproduce
- Response within 24-48 hours

Check Status Page:
- Visit status.m1ssion.eu for service status
- Subscribe to incident notifications
- Check scheduled maintenance times
