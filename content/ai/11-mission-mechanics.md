# Mission Mechanics

Understanding how M1SSION missions work will help you become a more effective treasure hunter.

## Mission Structure

### Mission Lifecycle
1. **Announcement**: Mission details revealed to premium tiers first
2. **Early Access Period**: Staggered access based on subscription tier
3. **General Release**: All tiers can participate
4. **Active Period**: Mission remains active for set duration
5. **Conclusion**: Winner announced, prizes awarded
6. **Archive**: Mission moves to Hall of Winners

### Mission Components

**Target Location**
- Physical location where treasure is hidden
- Can be indoor or outdoor
- Always in safe, publicly accessible areas
- Specific address revealed only after completion

**Clues**
- Progressive hints that narrow down location
- Released on schedule (varies by tier)
- Can be textual, visual, or riddles
- Some clues are premium-only

**BUZZ Zones**
- Proximity indicators without revealing exact location
- Show "warm/hot" feedback as you get closer
- Limited uses per day (tier-dependent)
- Resets daily at midnight

**BUZZ Map**
- Shows general area on map (radius varies)
- Premium feature (Gold tier and above)
- Updates as more clues are revealed
- Can be used strategically with BUZZ

## Clue Types

### Text Clues
Classic written hints that:
- Describe surroundings
- Reference landmarks
- Use wordplay or riddles
- May contain cultural references

### Visual Clues
Images that show:
- Nearby architecture
- Distinctive features
- Street-level views
- Historical photos

### Coordinate Clues
Numerical hints like:
- Partial coordinates
- Distance measurements
- Direction indicators
- Area boundaries

### Riddle Clues
Puzzle-based hints requiring:
- Lateral thinking
- Local knowledge
- Pattern recognition
- Research skills

## Scoring System

### XP (Experience Points)
Earned through:
- Finding clues: 10 XP each
- Using BUZZ: 15 XP per use
- Using BUZZ Map: 10 XP per use
- Daily check-ins: 5 XP
- Completing missions: 100 XP
- Weekly leaderboard placement: Up to 50 XP

### Leaderboard Points
Separate from XP, based on:
- Speed of completion
- Number of BUZZ uses (fewer is better)
- Final shot accuracy
- Weekly performance

### Badges
Earned for achievements:
- First mission completion
- Speed records
- Consecutive days playing
- Special events
- Subscription milestones

## Strategy Tips

### Efficient Clue Usage
- Read all available clues thoroughly
- Cross-reference multiple clues
- Research references you don't understand
- Share insights with team (if allowed)

### BUZZ Optimization
- Don't waste BUZZ too early
- Use after narrowing area with clues
- Strategic BUZZ Map usage saves attempts
- Consider daily limit before using

### Final Shot Timing
- Maximum 2 attempts per day
- Wait until confident
- Use all available information
- Consider weather and accessibility
- Plan visit during daylight if possible

## Rules & Fair Play

### Allowed
- Using clues and maps provided
- Researching references
- Visiting locations to scout
- Taking photos for memory
- Sharing excitement on social media

### Not Allowed
- Hacking or exploiting bugs
- Sharing exact answers
- Using GPS spoofers
- Accessing restricted areas
- Multiple accounts per person

### Penalties
- First offense: Warning
- Second offense: Temporary suspension
- Third offense: Permanent ban
- Severe violations: Immediate ban

## Mission Etiquette

- Respect private property
- Don't disrupt businesses
- Be courteous to other players
- Follow local laws and customs
- Leave no trace
- Report issues responsibly

## Troubleshooting Missions

**Clue Not Unlocking**
- Check your subscription tier
- Verify mission schedule
- Ensure app is updated
- Contact support if bug suspected

**Location Not Matching**
- Ensure location services enabled
- Wait for GPS to stabilize
- Try moving to open area
- Check for regional restrictions

**Can't Submit Final Shot**
- Verify daily attempt limit
- Check mission is still active
- Ensure proper location accuracy
- Try again in a few minutes
